We are a team of builders, marketers, and operators focusing on the intersection of **AI Automation**, **Performance E-commerce**, and **Modern Web Experiences**. 

Our philosophy is simple: build clean, highly functional tools that get out of the user's way. We value a minimalist, modern aesthetic—keeping our codebases and user interfaces as neat and organized as a well-designed workspace.

### 🔭 What we're building

Our repositories span open-source tools, foundational templates, and daily workflow automations:

* **Multi-Agent Systems:** Autonomous AI agents that handle repetitive operational tasks, content research, and cross-channel analysis.
* **Modern CRM & Data Hubs:** Tools to unify multi-channel e-commerce data (TikTok, Shopee, Facebook) into streamlined, actionable dashboards.
* **Web Foundations:** Lightweight, scalable starters for rapidly spinning up internal mini-apps.

### 🛠 Our Core Stack

We prefer a modern, agile tech stack that allows us to ship fast and scale easily:
* **Frontend:** Next.js, React, TailwindCSS
* **Backend & DB:** Supabase, PostgreSQL
* **Infrastructure:** AI-first, edge-ready deployments
